I. Vue d'Ensemble du Projet 
Vous devez développer et déployer une plateforme de gestion d'étudiants basée sur 
une architecture microservice moderne. Ce projet couvre les technologies actuelles du 
DevOps et du développement full-stack. 
Vous formerez des groupes de 5 étudiants pour ce projet. La note sera la même pour 
tous. Toute fois si lors de la présentation, je constate la non implication d’un étudiant 
par rapport aux autres membres de son groupe, sa note pourra être diminués en 
conséquence ! Vous pourrez choisir qui présente quelle partie du projet mais des 
questions seront posées aléatoirement à chaque membre de votre. 
Important   Un projet non fonctionnel est un projet non noté .

Les numéros de port sont donnés à titre indicatif, vous pouvez les adapter en fonction 
de votre système et bien documenter vos choix. Attention un projet non fonctionnel n’est 
pas noté. 
Pour les numéros de port hors PRD, ajouter +10 pour trouver le port correspondant. 
Exemple : Le port du backend Node JS est 4000 pour l’environnement de PROD, celui 
de l’UAT sera 4010 et le DEV 4020. Vous pouvez garder un serveur MySQL pour tous 
les environnements. PostgreSQL est autorisé comme serveur de base de données.
